BeagleLogic

A Google Summer of Code 2014 Project with BeagleBoard.org

Welcome to BeagleLogic!

BeagleLogic is a software suite that realizes a logic analyzer on the BeagleBone Black / BlueSteel-Basic / the original BeagleBone using the two Programmable Real-Time Units.

BeagleLogic supports:

  • Sample Rates upto 100 MHz
  • Upto 14 channels (with onboard eMMC disabled), 12 otherwise
  • Capture buffers upto 320 MB or more (depending on RAM usage and applications running)
